nginx 在项目部署过程中的操作

nginx 部署Vue多页面

nginx 部署Nuxt项目

  • nuxt buidl打包
  • 复制 .nuxt nuxt.config.js static node_modules（可以复制也可以重新install）到服务器上去

  • 当前文件夹下创建startscript.js

  • 一定要记着 npm install node-cmd 不然会报错

  • npm install pm2 -g 服务器上安装 pm2

  • pm2 start startscript.js 启动成功

  • nginx配置代理

注意： nuxt start 不可以用 ip直接访问(可能我不会设置吧)
